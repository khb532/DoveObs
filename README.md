## 🔗 구조

각 폴더는 독립적인 Obsidian 볼트입니다. 전체를 clone하여 각 주제별로 옵시디언에서 열어서 사용할 수 있습니다.

```
D:\Obsidian\
├── Unreal\          (Vault)
├── Niagara\         (Vault)
├── Optimization\    (Vault)
└── Tools\           (Vault)
```

## 📝 사용 방법

1. 이 저장소를 clone
2. Obsidian에서 각 폴더를 볼트로 열기
3. 볼트 간 크로스 링크를 통해 연결된 지식 탐색
